`internal/shared/` holds cross-cutting, framework-adjacent utilities the hexagonal contexts all depend on (per [backend/index.md](index.md), this is the one layer domain code may import from besides stdlib). **config** (`config.go`): a single `Config` struct loaded via `caarlos0/env` + `godotenv`, with `env:"..."`/`envDefault:"..."` tags for every setting (DB/Redis URLs, Supabase JWT/JWKS, provider API keys, feature flags like `TailDemotionEnabled`, `BehavioralRankingEnabled`, `ExplorationEnabled`/`ExplorationRate`, `IdentityVerifyOnPersist`, and the detail/discography pair `DetailIdentityFirst`/`DiscographyV2` — both default **on** since 2026-07-23 (the identity-first V2 discography is the shipped default; the flags remain as env kill-switches to the single-provider path)). `validate()` fails fast if `SUPABASE_JWT_JWKS_URL` is unset. `LogValue()` implements `slog.LogValuer` to redact secrets, logging only booleans (`has_redis`, `has_lastfm`, etc.).

**database** (`database.go`): `NewPool` wraps `pgxpool`, pinging once at startup and **erroring when `databaseURL == ""`** — the database is required (unlike Redis), so an unconfigured URL fails startup instead of handing repositories a nil pool that panics on first query (structure-audit F2, 2026-07-16; callers wanting a "not configured" report check `cfg.DatabaseURL` themselves, as `health_check.go` does). **redis** (`redis.go`): `NewClient` returns `nil` only when the URL is empty/invalid; a ping failure is logged as a warning (`slog.Warn`) but the client is still returned (non-nil, network-unreachable) — so downstream cache adapters must tolerate failing Redis calls, not just check for a nil client, when Redis is configured but unreachable (see [cache-layer](discovery/cache-layer.md)).

**logging** (`logging.go` + `ringbuffer_handler.go`): `Setup(logLevel, development)` takes the two values it needs (not `*config.Config`, so logging has no intra-shared dependency) and installs either a custom color `prettyHandler` (dev; renders `WithGroup` groups as dot-joined key prefixes) or `slog.NewJSONHandler` (prod) wrapped in a `ringHandler` that tees every record into a `RingBuffer` (fixed capacity `logRingCapacity = 1000`, capture floor `slog.LevelDebug` regardless of stdout's level) feeding the Mission Control logs panel (see [admin](admin/index.md)); `Subscribe` gives a lossy live-tail channel (slow subscribers drop records, never block the logger).

**httptrace** (`recorder.go`/`replay.go`): `Recorder` is a debug-only `http.RoundTripper` that buffers full request/response bodies per exchange (never for production traffic — the file doc comment is explicit); `Replayer` answers requests from previously recorded `Exchange`s matched on `(method, URL, body)` in FIFO capture order, erroring loudly on an unmatched request — enables deterministic offline discovery-eval reruns (see [eval-harness](discovery/eval-harness.md)).

**httputil** (`errors.go`/`middleware.go`): `StatusError` interface (`error` + `HTTPStatus() int`) satisfied structurally by domain errors so `domain/` never imports `net/http`; `HandleServiceError` is the single domain-error→HTTP translation point. Middleware: `CorrelationID` (stamps an 8-char id via context + response header), `RequestLogger`, `Recoverer` (panic recovery → structured log + 500), `MaxBodySize`.

**textnorm** (`normalize.go`/`fuzzy.go`): `NormalizeForMatch` canonicalizes titles/names (NFKC → lowercase → strip diacritics → drop bracketed suffixes → `&`→"and" → strip apostrophes → Unicode-aware symbol stripping via `stripSymbols` that preserves non-ASCII letters/numbers, e.g. CJK — an explicit `AIDEV-NOTE` documents a past bug where the old ASCII-only `\w` regex zeroed out CJK titles). `TokenSortRatio`/`LevenshteinDistance` in `fuzzy.go` implement rapidfuzz-style token-sort fuzzy matching for discovery dedup.

**UserId** (`userid.go`): the canonical cross-context identity value object — unexported `uuid.UUID` field, `NewUserId`/`ParseUserId` constructors, `UUID()`/`String()`/`IsZero()` accessors — used by every domain type (`Track.UserId`, `Playlist.UserId`, `QueueState.UserId`, etc.) per the domain-layer rule that shared value objects — plus the pure `textnorm` functions — are the only permitted domain-layer imports from `internal/shared/`. That rule is now lint-enforced: depguard (`services/go-api/.golangci.yml`, run in the test-backend CI workflow) fails the build if a `domain/` package imports any other shared subpackage or if anything under `internal/shared/` imports a feature.
